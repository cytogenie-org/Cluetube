Building a Tree Table

A tree-table displays large amounts of data in a way that you design. Construct a tree-table by clicking the tree icon, to open the tree-to-table menu-->build (located above the table on the right side). There are several options. The most common way is to click the first option "Drag and drop". When Drag and drop is selected, a blank tree panel appears on the left of the table. Select the column header, with left mouse button down, and drag the column to the left and drop the column in the tree panel. Continue to build the tree by dragging a column to the left and dropping it on the tree panel. The Tree-to-Table menu gives you the option to save your tree to your favorites list.

Tooltips

Cytogenie has many tooltips throughout the interface. When you pause the mouse over icons and user-interface widgets for a few seconds, a tooltip will appear for several seconds. There are also tooltips when a menu item is not active that explains why the feature is not active. Just hoover the mouse over the feature and a tooltip will appear. Tooltips appear when you hoover over grayed-out features as well.

Common Icons

  • Move the selected line(s) in the table up one row per click.
  • Move the selected line(s) in the table down one row per click.
  • Create a tree from the table or restore the table from the tree.
  • Create a new item. The item to be created is based on the context.
  • Previously created items. See a description of the previously created items in this context and select one for this location.
  • Select a row in the table and click the trashcan to delete the item from the list. If the data in the row has not been defined in a knowledge base that you have privileges to change, you will not be able to delete the information.
  • See all the information in the selected row in a separate window.
  • Open the description of the selected item in the protégé form that defines it.
  • Tools is a context-sensitive menu on the tables with many common and specific data object manipulations. The menu options are also available by clicking the right mouse button when the mouse cursor is inside of the table. On Mac, if the right mouse button has not been assigned, use Control-Click to activate the menu.

File Resources

File Resources create a folder to store the FCS data, the pipetting guide, and any other files you wish to associate with this study. The alias is just a tag that will be added to the CytoGenie-created files in order to distinguish the different parts of a study should you wish to put more than one protocol in the same folder. For example CytoGenie creates the pipetting guide that it names “pipettingGuide” concatted with the alias concatted with “.pdf”. You can set up the folder and the alias when you create the protocol.

The Tools Menu

The Tools Menu is a context-senstive menu located on the far right of the icon bar on most pages. The menu items on the top part of the menu are specific to the page that is currently showing. The lower half of the menu has more common options that are performed on tables. These menu actions are useful when configuring the table views to your liking. The configurations will be saved in your preferences.

  • Manage columns: various options to sort the contents of a columns, change the display order of the columns and hide a column.
  • Sort/arrange columns: all the possible columns for this table are listed in a popup window. You can show/hide particular columns, change the order of the columns, and determine the sort order of the table.
  • Manage rows: selected rows can be displayed in a browser, import and export the table contents.
  • Manage view
  • Put in separate window: show the table in its own window
  • Tree: create a tree table.

Filter by Knowledge Base Level

Several of the lists, for example instruments, reagents, and protocols, can be filtered by the knowledge base that contains it. The choices are Mine, My Lab, Facility, and Inactive. Inactive items are those that will not be available to be used in the future, but may be referenced in prior protocols. When Mine is selected on the reagents page, the reagents defined in your researcher knowledge base will be displayed and those reagents defined at your lab containing knowledge base will not be displayed. Select the biggest, containing knowledge base to see more (usually the facility level).
